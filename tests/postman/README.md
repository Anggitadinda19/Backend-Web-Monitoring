# Postman Test Collection

Folder ini berisi file yang digunakan untuk menjalankan API testing
menggunakan Postman + Newman.

## File

- **collection.json** → Kumpulan request API
- **environment.json** → Environment (base_url)
- **newman-report.html** → Laporan hasil testing otomatis (dihasilkan oleh Newman)
